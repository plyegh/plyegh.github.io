학습용으로 제공되는 워게임 문제를 통해, 사용자 입력이 검증 없이 쿼리에 이어붙을 때 어떤 위험이 생기는지 확인했다.

## 안전한 코드로 고치기

```python
cursor.execute(
  'SELECT * FROM users WHERE name = %s', (name,)
)
```

핵심은 문자열 연결 대신 파라미터 바인딩을 쓰는 것이다. 문제를 푸는 것에서 끝내지 않고, 같은 취약점을 실제 코드에서 어떻게 예방하는지까지 함께 기록해 두려 한다.
